# CS290-IKKEditor
CS290 Final Project. IKKEditor is a simple photo editor that allows you to post anonymously onto a shared gallery with other users.
Editor Features include:
* Painting
* Erasing
* Adjustments
* Cropping
* Decorations
* Posting to cloud gallery
